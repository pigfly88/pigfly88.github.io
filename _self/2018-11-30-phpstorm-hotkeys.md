双击 Shift 或者 Ctrl + N # Navigate, 根据名字快速导航到类或方法

Ctrl + Alt + 左/右箭头 # 返回/前进到上次导航操作

Ctrl + F12 # 显示类结构

Ctrl + B # 跳转到函数的声明处

Alt + 上/下箭头 # 上下切换函数

Alt + F7 # 查看函数/变量在哪里被使用

Ctrl + Shift + 箭头 # 移动行

Ctrl + G # 跳转到某一行

Ctrl + K # Git commit

Ctrl + Shift + ` # Git checkout

Ctrl + Tab # 切换标签页

Ctrl + E # 显示最近打开的文件列表

Ctrl + F4 # 关闭当前标签页
